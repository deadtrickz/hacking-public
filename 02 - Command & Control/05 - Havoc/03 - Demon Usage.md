	# Havoc Demon Usage
- Metasploit has Meterpreter, Cobalt Strike has Beacon, Havoc has Demon.

### Customize Demon Behavior
```
help config
```

## Enumerate with Demon
- use `shell [Native_Command]` to send commands to use non-demon commands.
	- `shell whoami`

```
whoami
```
```
ipconfig
```
```
sc_enum
```
```
dcenum
```
